# SCT_DA_4
Business Insights Report and Exploratory Data Analysis (EDA) on Bank Marketing Campaign Dataset using Python, Pandas, Matplotlib and Seaborn.

Introduction
This project was completed as part of the Data Analytics Internship at SkillCraft Technology.
The objective was to perform Exploratory Data Analysis (EDA) on a Bank Marketing Campaign dataset and derive business insights that can help improve marketing effectiveness and increase term deposit subscriptions.

Project Objective
Analyze customer demographics, campaign performance, and communication strategies to identify factors that influence term deposit subscriptions.
Dataset Features
The dataset contains customer information such as:
- Age
- Job
- Marital Status
- Education
- Account Balance
- Contact Method
- Campaign Interactions
- Previous Campaign Outcome
- Deposit Subscription Status

Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab
- GitHub

Project Workflow
1. Data Loading
2. Data Exploration
3. Data Cleaning
4. Exploratory Data Analysis
5. Visualization Creation
6. Business Insight Generation
7. Recommendation Development

Visualizations Included
- Deposit Subscription Distribution
- Age Distribution
- Job vs Deposit Analysis
- Education vs Deposit Analysis
- Balance vs Deposit Analysis
- Contact Method Analysis
- Campaign Performance Analysis
- Correlation Heatmap

Key Insights
- Customer characteristics significantly impact subscription decisions.
- Account balance shows a relationship with deposit subscriptions.
- Certain contact methods perform better than others.
- Previous campaign success improves future conversion probability.
- Campaign frequency should be optimized to avoid diminishing returns.
